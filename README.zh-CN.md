# 我的 Skills

[English](README.md)

Some skills using in my life.

这里存放我在日常生活和工作中会用到的 Codex skills。每个 skill 都尽量沉淀成可复用的流程、模板和注意事项，而不是一次性的临时脚本。

## Skills

### a-share-etf-permanent-portfolio-backtest

用于回测 A 股 / 中国 ETF 投资组合，按每月最后一个交易日收盘价重新平衡到目标权重。

这个 skill 适合做 ETF 组合研究，主要能力：

- 通过 AKShare 获取 ETF 日线收盘价；
- 默认使用 `qfq` 前复权价格，更适合分红再投入/总收益口径分析；
- 支持本地宽表 CSV，便于使用同花顺、券商或数据商导出的价格；
- 自动归一化目标权重，并按月末再平衡；
- 当某只 ETF 历史数据短于请求区间时，报告实际可回测起点；
- 生成中文报告，包括每月收益率、每年收益率、汇总说明和收益贡献拆解；
- 只做教育性组合诊断，不给买卖建议。

### travel-plan-workbook

用于创建、编辑和复用旅行计划 Excel 工作簿。

这个 skill 会先确认真实可用的旅行时间，尤其是日期范围是否包含去程和返程航班日，再生成行程表。它适合制作包含路线、住宿、预约、预算和风险提示的实用型旅行计划。

主要能力：

- 按天生成旅行行程安排；
- 生成并插入路线图图片，并做视觉验收；
- 记录酒店、预约、闭馆风险和来源链接；
- 估算预算、现金和刷卡比例；
- 通过 `assets/template.xlsx` 复用意大利风格 Excel 模板；
- 保持工作簿文字和路线图图片文字语言一致；
- 避免把被拦截的地图瓦片、错误页或异常图片插入最终工作簿。
